# nystreets

Adds NPCs walking around like on the streets of New York.

## Features

- **Global Spawner**: Spawns NPCs dynamically around players up to a configurable limit.
- **`clear_multitud` Chat Command**: Allows server admins to quickly remove all `nystreets:npc` entities from the world.
- **NPC Spawn Egg (`nystreets:egg`)**: Allows players to manually spawn NPCs where needed.
